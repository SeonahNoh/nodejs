# ToD App 
## 🌱 목적 
- 서버 프로그램을 사용하여 ToDo App 구현 

## 🌱 기술 스택
- Node.js 
- MongoDB Altas
- Bootstrap
- EJS 

## 🌱 특징
1. Todo Item을 추가, 삭제, 수정할 수 있는 웹 페이지
2. 로그인, 마이페이지 지원
3. 로그인 기능은 session-based authentication 방식으로 구현
